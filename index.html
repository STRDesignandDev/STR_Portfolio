The Windows and Walls App Redesign

The app is a graphics ordering system with a client facing front end, and administrative job tracking system. It is used primarily by the beverage industry to place POS advertisements graphics in retail locations.

The Old Windows And Walls App

The legacy version of the site was built exclusively for one client, and had a very rigid and out of date structure. A decision was made to create a new version of the app in a modern JS framework (VUE JS). This gave myself and the team an incredible opportunity to give an already highly utilized application a complete makeover.

Ideation and Architecture Restructure

I began reverse engineering the old application. Breaking down the user flows, restructuring the site architecture, and identifying redundancy. Utilizing user feedback and UX best practices, I began prototyping in low fidelity and annotating instructions for the dev team.

UX Problem

The legacy system did not allow for onboarding of new clients. Each client needs to have unique pricing, messaging, and asset management.

UX Solution

Design an admin backend that allows for full control of multiple clients. Create the ability for internal users to manage clients, options, pricing, and more through the interface. This greatly reduced the need for costly development time.

UX Problem

The legacy system had a rigid user flow, users had to go through a strict set of sequential steps to submit an order, and could not revert back through decisions or mitigate errors

UX Solution

Create a user flow with more freedom and happy paths. Enable/disable interactive features to guide the user through the process. Provide better error recognition. Condense the UI to minimize the need for scrolling
